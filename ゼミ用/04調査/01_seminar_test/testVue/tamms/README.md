# tamms

### npm
```
npm install
```

### Vue
```
npm install -g @vue/cli
npm install vue-router
vue create tamms
```

### 起動
```
npm run serve
```

### ビルド
```
npm run build
```

### axios
```
npm install axios vue-axios
```

### axios-cookiejar-support（cookie持続的保持）

```
npm install tough-cookie axios-cookiejar-support
```

```
npm install bootstrap-vue
```
